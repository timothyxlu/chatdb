MCP Server
Connect AI clients to your chat history via the Model Context Protocol
Active
🔌 ChatDB exposes an MCP (Model Context Protocol) server over HTTP+SSE. AI applications — Claude, ChatGPT, Gemini — connect here to save conversations as they happen and search past context at the start of new ones. Authenticate each app with a personal API token below.
Server URL
Add this to your MCP client config
https://chatdb.timothyxlu.com/mcp
Transport
Protocol version
HTTP + SSE · MCP 2025-03-26
Client config snippet
Paste into claude_desktop_config.json or your MCP client settings
claude_desktop_config.json
{
  "mcpServers": {
    "chatdb": {
      "type": "sse",
      "url": "https://chatdb.timothyxlu.com/mcp/sse",
      "headers": {
        "Authorization": "Bearer chatdb_tk_••••••••••••3f9a"
      }
    }
  }
}
